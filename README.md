Introduction
This Unlocker is designed for Fluffy-game (versions 1.0–3.5) and enables:

Unlocking hidden features and levels
Accessing characters/skins not available in the base game
Adding custom settings via fluffy_unlocker.cfg
Tested on:

Windows 10/11 (x64)
Linux (Ubuntu 22.04+, Fedora 38+)
macOS (Intel & Apple Silicon — via Unity Player)
⚠️ Before running the Unlocker — close Fluffy-game completely and ensure no background processes like Fluffy-game.exe are running.

Prerequisites
Unity Player (if using .exe or .AppImage)
.NET 6+ or Mono 6.12+ (for Linux)
Optional: Visual Studio or Rider (for manual script editing)
Limitations
If the game crashes after using the Unlocker:

+-----------------------------------------------------------------------------+ | IMPORTANT: | | ========== | | | | If the game crashes after unlocking: | | | | 1. Try running the game in Compatibility Mode (Windows) or with --force-opengl. | | 2. Edit fluffy_unlocker.cfg and disable experimental features one by one. | | | +-----------------------------------------------------------------------------+

Windows
Run the Unlocker as Administrator:

FluffyUnlocker_Win.exe — applies unlock patches
FluffyRestore_Win.exe — reverts to original game state
FluffyUpdateTools.exe — downloads latest modded assets or configs
Right-click → “Run as administrator” or use Command Prompt with elevated privileges.

Linux
Run with sudo or as root:
